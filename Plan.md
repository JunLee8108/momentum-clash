# 카드 RPG 게임 디자인 문서 (GDD)

> **작업명**: Momentum Clash
> **장르**: TCG (Trading Card Game) / 턴제 카드 배틀
> **플랫폼**: iOS (SwiftUI)
> **핵심 컨셉**: 속성 지형 쟁탈 + 기세(Momentum) 자원 관리 전략 카드 배틀

---

## 1. 게임 개요

속성 지형 시스템과 기세(Momentum) 게이지를 중심으로 한 공격 지향 전략 카드 배틀 게임. 플레이어는 매 턴 기본 기력과 전투로 쌓은 기세를 자원으로 활용하여 카드를 소환하고, 필드 지형을 장악하며, 기세 스킬을 발동하는 역동적인 플레이를 경험한다.

### 핵심 차별점
- **기세 = 자원**: 기세 게이지가 소환 비용이자 스킬 자원. 공격적 플레이가 더 많은 선택지를 만든다
- **속성 지형**: 필드 5슬롯이 전투 중 실시간으로 속성 변화하며, 지형 장악이 승리의 핵심
- **통합 필드**: 몬스터와 마법이 같은 5슬롯을 공유하여 공간 관리 전략 발생
- **선택 드로우**: 매 턴 2장을 보고 1장을 선택하여 운 요소 감소

### 디자인 철학
- 공격이 최선의 방어: 수비만으로는 자원(기세)이 쌓이지 않아 점점 불리해짐
- 역전 가능성: 밀리는 쪽에게 복귀 장치를 제공하여 마지막까지 긴장감 유지
- 간결한 전투, 깊은 전략: 전투 계산은 단순(전투력 비교)하지만 자원 배분과 지형 운용에서 깊이 발생

## 2. 핵심 시스템

### 2.1 기세(Momentum) 시스템 ★핵심★

기세는 이 게임의 핵심 자원이다. 카드 소환 비용이자 강력한 기세 스킬의 발동 자원이다.

**자원 구조**:
```
매 턴 사용 가능한 자원 = 기본 기력(2) + 보유 기세(0~10)
```
- 기본 기력: 매 턴 시작 시 2를 무조건 받음 (누적 불가, 그 턴에 안 쓰면 소멸)
- 기세: 전투와 행동으로 쌓이며, 턴 간 누적됨 (최대 10)
- 카드 소환 시 기본 기력을 먼저 소모하고, 부족하면 기세를 추가 소모

**기세 획득**:
| 행동 | 기세 |
|------|------|
| 몬스터로 공격 성공 (상대 몬스터 전투) | +1 |
| 상대 몬스터 파괴 | +1 |
| 상대 LP에 직접 데미지 | +2 |
| 속성 지배 달성 중 (매 턴) | +1 |
| 피격 시 (내 몬스터가 파괴됨) | +1 |

**위기 보정**:
- LP가 50% 이하: 기본 기력 2 → **3**으로 증가
- LP가 25% 이하: 기본 기력 2 → **4**로 증가

**기세 감소**:
| 상황 | 기세 |
|------|------|
| 턴 종료 시 공격을 하지 않음 | -1 |
| 턴 종료 시 자연 감소 | -1 (기세 6 이상일 때만) |

> 자연 감소는 기세를 무한히 비축하는 것을 방지한다. 쌓으면 써야 한다.

**기세 스킬** (기세만 소모하여 발동, 기본 기력 사용 불가):
| 스킬 | 기세 비용 | 효과 |
|------|-----------|------|
| 투지 | 3 | 이번 턴 몬스터 1체 전투력 +500 |
| 지형 전환 | 4 | 필드 슬롯 2개를 원하는 속성으로 변경 |
| 연속 공격 | 5 | 몬스터 1체가 이번 턴 2회 공격 가능 |
| 전선 돌파 | 6 | 이번 턴 모든 몬스터 전투력 +300 |
| 기세 폭발 | 8 | 상대 필드 몬스터 전체에 기세 수 × 100 데미지 |
| 완전 각성 | 10 | 필드 위 몬스터 1체를 각성 형태로 변환 (특수 카드 한정) |

**소환과 스킬의 딜레마** (게임의 핵심 판단):
```
기세 7 보유 시 선택지:
  A) 비용 5 에이스 몬스터 소환 (기력2 + 기세3 소모) → 기세 4 남음
  B) 비용 2 몬스터 소환 (기력2 소모) + 연속 공격 스킬 (기세5) → 기세 2 남음
  C) 소환 안 하고 기세 폭발(8) 발동 → 상대 필드 정리 → 기세 거의 소진
```

### 2.2 속성 시스템

6개 속성이 상성 관계를 가진다.

| 속성 | 강함 (→) | 약함 (←) | 색상 |
|------|----------|----------|------|
| 🔥 화(Fire) | 풍 | 수 | 빨강 |
| 💧 수(Water) | 화 | 뇌 | 파랑 |
| 🌿 풍(Wind) | 지 | 화 | 초록 |
| ⛰️ 지(Earth) | 뇌 | 풍 | 갈색 |
| ⚡ 뇌(Thunder) | 수 | 지 | 노랑 |
| 🌑 암(Dark) ↔ ✨ 광(Light) | 서로 상극 | 서로 상극 | 보라/흰색 |

**상성 효과**:
- 유리 속성으로 공격 시: 데미지 1.3배
- 불리 속성으로 공격 시: 데미지 0.7배
- 광 ↔ 암: 서로 1.5배 (하이리스크 하이리턴)

### 2.3 속성 지형 시스템

필드는 **5개 통합 슬롯**으로 구성되며, 각 슬롯은 속성 지형을 가진다.

```
  플레이어 필드:
  [ 슬롯1 ] [ 슬롯2 ] [ 슬롯3 ] [ 슬롯4 ] [ 슬롯5 ]
    화염검사   지속마법    (빈칸)   안개정령    (빈칸)
    지형:🔥   지형:🔥    지형:중립  지형:💧   지형:중립
```

**지형 변화 규칙**:
- 몬스터를 소환하면, 해당 슬롯이 그 몬스터의 속성으로 오염된다
- 마법 카드 중 "지형 마법"으로 직접 지형을 변환할 수 있다
- 카드가 파괴/제거되어도 지형 속성은 1턴 동안 유지된다

**지형 보너스**:
- 같은 속성 지형 위의 몬스터: 전투력 +200
- 3개 이상 같은 속성 지형 (**속성 지배**): 해당 속성 카드 전체 전투력 +300, 기세 +1 (매 턴)
- 5개 전부 같은 속성 (**완전 지배**): 해당 속성 카드 전투력 +500, 상대 몬스터 전체 전투력 -200

**통합 필드의 전략적 의미**:
- 몬스터와 마법이 같은 5슬롯을 공유 → "몬스터를 더 깔까, 지속 마법을 유지할까"
- 지속 마법도 슬롯을 차지하므로 지형에 영향 (마법 카드의 속성이 지형을 결정)
- 5칸이라는 제한이 모든 판단에 긴장감을 부여

## 3. 카드 체계

### 3.1 카드 공통 요소

모든 카드가 가지는 공통 속성:
| 요소 | 설명 |
|------|------|
| 이름 | 카드 이름 |
| 속성 | 7속성 중 1개 (화/수/풍/지/뇌/암/광) |
| 비용 | 소환에 필요한 자원 (1~5) |
| 레어리티 | N / R / SR / UR |
| 아트워크 | 카드 일러스트 |

### 3.2 카드 종류

#### ⚔️ 몬스터 카드
전투의 핵심. 필드 슬롯에 소환하여 전투를 수행한다.

| 요소 | 설명 |
|------|------|
| 전투력(CP) | Combat Power. 공격과 방어 모두에 사용 (0~3500) |
| 타입 | 전사, 마법사, 드래곤, 기계, 정령, 언데드 등 |
| 효과 | 없음(일반) 또는 효과 텍스트 |

**몬스터 분류**:

**일반 몬스터**:
- 효과 없음. 같은 비용의 효과 몬스터보다 전투력이 높음
- 지형 보너스 + 높은 기본 전투력으로 순수 전투에서 강력
- 전용 서포트 카드 존재 ("일반 몬스터를 대상으로~")

**효과 몬스터**:
- 소환 시 / 파괴 시 / 매 턴 등 다양한 타이밍에 효과 발동
- 일반 몬스터보다 전투력이 낮지만 전략적 가치가 높음

**각성 몬스터** (Phase 5 확장):
- 기세 스킬 "완전 각성"으로 특정 몬스터에서 변환
- 덱에 직접 넣을 수 없음

**비용별 전투력 기준**:

| 비용 | 일반 몬스터 CP | 효과 몬스터 CP | 위치 |
|------|---------------|---------------|------|
| 1 | 400~600 | 200~400 | 초반 벽, 지형 오염용 |
| 2 | 800~1200 | 600~900 | 초반 주력 |
| 3 | 1400~1800 | 1100~1500 | 중반 주력 |
| 4 | 2000~2500 | 1700~2100 | 상위 전력 |
| 5 | 2700~3200 | 2300~2800 | 에이스급 |

> 기본 기력(2)만으로는 비용 2까지만 소환 가능.
> 비용 3 이상은 기세를 투자해야 하므로, 강한 카드를 내려면 공격적 플레이가 필수.

#### 🔮 마법 카드
다양한 효과를 적용한다. 필드에 놓으면 슬롯을 차지한다.

| 마법 유형 | 비용 | 설명 |
|-----------|------|------|
| 일반 마법 | 1~3 | 사용 즉시 효과 발동 후 묘지로 (슬롯 차지 안 함) |
| 지속 마법 | 2~4 | 슬롯에 배치, 매 턴 효과 적용. 슬롯 차지 |
| 장착 마법 | 1~3 | 몬스터 1체에 장착, 전투력 변경. 슬롯 차지 안 함 |
| 지형 마법 | 2~3 | 필드 슬롯의 속성을 변경. 즉시 발동 후 묘지 |

> 일반 마법과 지형 마법은 발동 후 묘지로 가므로 슬롯을 차지하지 않는다.
> 지속 마법은 슬롯을 차지하므로 몬스터 배치 공간이 줄어드는 트레이드오프가 있다.

### 3.3 방어막(Shield) 시스템

전투력 단일 스탯 체계에서 수비 플레이를 가능하게 하는 보조 시스템.

- 방어막은 일부 카드 효과로 몬스터에게 부여됨 (예: "아군 몬스터 1체에 방어막 500 부여")
- 전투 시 데미지를 방어막이 먼저 흡수
- 방어막은 턴 종료 시 소멸 (일회성)
- 방어막이 있어도 전투력 차이로 파괴는 될 수 있음

```
예시:
  내 몬스터 CP 1200 + 방어막 500
  상대 몬스터 CP 1600으로 공격
  
  방어막 500 소모 → 남은 데미지 1600 - 500 = 1100
  1100 < 1200(내 CP) → 파괴되지 않음, LP 데미지 없음
  
  방어막 없었다면:
  1600 > 1200 → 내 몬스터 파괴, LP 400 데미지
```

## 4. 대전 구조

### 4.1 기본 규칙

| 항목 | 값 |
|------|-----|
| 시작 LP (Life Points) | 8000 |
| 시작 패 | 5장 |
| 덱 매수 | 40장 (최소 30, 최대 50) |
| 같은 카드 최대 | 3장 |
| 통합 필드 슬롯 | 5칸 (몬스터 + 마법 공유) |
| 기본 기력 (매 턴) | 2 (LP 50% 이하 시 3, LP 25% 이하 시 4) |
| 기세 초기값 | 선공: **2** / 후공: 0 (최대 10) |

### 4.1.1 선공/후공 밸런스

선공과 후공이 서로 다른 이점을 갖도록 설계한다.

| 요소 | 선공 | 후공 |
|------|------|------|
| 시작 기세 | **2** | 0 |
| 첫 드로우 | 2장 중 1장 선택 | 2장 중 1장 선택 |
| 첫 턴 공격 | **불가** | **가능** |

**설계 의도**:
- 선공 이점: 기세 2로 시작하여 턴 1에 비용 3~4 카드 소환 가능 → 강한 보드 선점
- 후공 이점: 첫 턴부터 공격 가능 → 기세 획득 기회 + 선공 보드에 즉시 대응
- 결과: 선공은 "보드 우위", 후공은 "템포 우위"로 균형

### 4.2 턴 진행

```
1. 드로우 페이즈
   → 덱에서 2장을 본 후 1장 선택, 나머지 1장은 덱 맨 아래로
   → (첫 턴 포함 모든 턴 동일하게 선택 드로우)

2. 스탠바이 페이즈
   → 기본 기력 수령 (2 / 3 / 4)
   → 지속 효과 처리
   → 속성 지배 중이면 기세 +1
   → 기세 6 이상이면 자연 감소 -1

3. 메인 페이즈
   → 몬스터 소환 (비용 지불: 기력 우선, 부족 시 기세 소모)
   → 마법 카드 사용 (비용 지불)
   → 기세 스킬 사용
   → 순서 자유, 자원이 허용하는 한 여러 번 가능

4. 배틀 페이즈
   → 몬스터로 공격 선언 (첫 턴 선공 불가)
   → 각 몬스터는 턴당 1회 공격 (기세 스킬 "연속 공격" 제외)
   → 전투 처리 후 기세 획득/감소 즉시 반영

5. 엔드 페이즈
   → 미사용 기본 기력 소멸
   → 방어막 소멸
   → 패 8장 초과 시 버림
   → 공격을 하지 않았다면 기세 -1
```

### 4.3 전투 계산

전투력(CP) 단일 비교:

```
내 몬스터 vs 상대 몬스터:
  → 전투력 비교
  → 낮은 쪽 파괴
  → 차이만큼 패배자의 LP 데미지
  → 동일하면 양쪽 다 파괴

직접 공격 (상대 필드에 몬스터 없을 때):
  → 공격 몬스터 전투력만큼 상대 LP에 직접 데미지
```

**전투력 계산 순서**:
```
1. 기본 전투력 (카드 CP)
2. + 장착 마법 / 지속 효과
3. + 방어막 (데미지 흡수, 전투력 자체를 올리지는 않음)
4. + 지형 보너스 (+200 / +300 / +500)
5. × 속성 상성 배율 (0.7 / 1.0 / 1.3 / 1.5)
6. + 기세 스킬 효과 (해당 시)
```

> 방어막은 전투력에 더해지지 않고, 받는 데미지를 흡수하는 별도 계층.

### 4.4 승리 조건
- 상대 LP를 0으로 만든다
- 상대 덱이 0장이 되어 드로우할 수 없다
- 특수 승리 조건 카드 (추후 확장)

## 5. 전략 아키타입 (예상 덱 유형)

| 아키타입 | 전략 | 핵심 |
|----------|------|------|
| 화염 러시 | 저비용 화 몬스터 대량 소환 → 빠른 지형 지배 → 기세 폭주 | 속도, 기세 축적 |
| 대지 요새 | 높은 CP 지 몬스터 + 방어막 카드 → 버티면서 에이스 소환 | 방어막, 후반 에이스 |
| 뇌격 컨트롤 | 효과 몬스터로 상대 지형 파괴 + 속성 전환 | 지형 방해 |
| 수풍 순환 | 수/풍 혼합으로 상성 커버 + 유연한 지형 운용 | 복합 속성, 유연성 |
| 광암 하이리스크 | 광/암 카드의 1.5배 상성으로 극단적 화력 | 높은 데미지, 높은 리스크 |
| 바닐라 비트 | 일반 몬스터 위주 + 지형 보너스 극대화 | 높은 기본 CP, 안정성 |

## 6. 게임 모드

### 6.1 스토리 모드 (싱글)
- 챕터별 스토리 진행
- 각 챕터 보스는 특정 속성/전략 특화
- 클리어 보상: 새 카드, 카드팩, 게임 내 재화
- 특수 룰 스테이지 (예: "기세 시작값 5", "지형 매 턴 랜덤 변환")

### 6.2 프리 배틀 (싱글)
- AI 상대 자유 대전
- AI 난이도: 쉬움 / 보통 / 어려움 / 극한
- AI 덱 타입 선택 가능

### 6.3 대전 모드 (멀티 - Phase 5 이후)
- 랭크 매치 / 프리 매치
- 시즌제 랭킹

### 6.4 덱 빌더
- 보유 카드 목록에서 덱 구성
- 속성/타입/비용 필터
- 덱 분석: 속성 비율, 평균 비용, 몬스터/마법 비율, 비용 커브 그래프
- 덱 슬롯 여러 개 저장

### 6.5 카드 수집
- 카드팩 개봉 (재화로 구매)
- 레어리티: 일반(N) → 레어(R) → 슈퍼레어(SR) → 울트라레어(UR)
- 카드 도감: 수집률 확인
- 중복 카드 분해 → 재화 → 원하는 카드 제작

## 부록: 용어 정리

| 용어 | 설명 |
|------|------|
| LP | Life Points, 생명력 (시작 8000) |
| CP | Combat Power, 전투력 |
| 기세 | Momentum 게이지 (0~10), 소환 자원 겸 스킬 자원. 선공 시작값 2 |
| 기본 기력 | 매 턴 무조건 받는 자원 (2, 위기 시 3~4) |
| 방어막 | Shield, 데미지를 흡수하는 일회성 보호막 |
| 속성 지배 | 같은 속성 지형 3개 이상 확보 |
| 완전 지배 | 같은 속성 지형 5개 전부 확보 |
| 각성 | 기세 스킬로 몬스터를 상위 형태로 변환 (Phase 5) |
| 선택 드로우 | 2장을 보고 1장을 골라 가져가는 드로우 방식 |
| 통합 필드 | 몬스터와 마법이 같은 5슬롯을 공유하는 필드 구조 |
