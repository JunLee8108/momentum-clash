# AI 전면 개선 계획 — "완벽한 AI"

## 현재 문제점 분석

### 1. 불리한 전투에 무조건 돌격 (가장 큰 문제)
- `chooseBestTarget()`의 **Tier 3**: 이길 수 없어도 무조건 가장 약한 적을 공격
- 결과: 전투력 500 몬스터가 전투력 1200 몬스터를 공격 → AI 자멸

### 2. 공격 순서 최적화 없음
- `monsterSlotIndices` 순서(슬롯 0→4)로 공격
- 속성 유리한 몬스터가 뒤에 있으면 앞의 불리한 몬스터가 먼저 공격

### 3. 소환 전략 부재
- 상대 필드 속성을 고려하지 않음 (상성 유리한 몬스터 우선 소환 X)
- 지형 시너지를 고려하지 않음 (같은 속성 지형 슬롯에 소환 X)

### 4. 기세 스킬 미사용
- AI가 기세 스킬(`MomentumSkill`)을 전혀 사용하지 않음
- 기세가 쌓여도 활용하지 못함

### 5. 드로우 판단 단순
- 현재 보드 상황과 무관하게 CP + 바닐라/효과 보정만으로 선택

## 개선 계획

### Phase 1: 전투 판단 개선 (chooseBestTarget + 공격 순서)

**파일**: `BasicAI.swift` — `chooseBestTarget()`, `planBattlePhase()`

#### 1-1. 불리한 전투 회피
- Tier 3 제거 → 이길 수 없으면 **공격하지 않음** (`nil` 반환)
- Tier 2 조건 강화: 동귀어진은 **상대 몬스터가 고가치(CP ≥ 800)일 때만** 허용

#### 1-2. 타겟 우선순위 재설계
```
1순위: 확실한 킬 (atkCP > defCP) — LP 데미지가 가장 큰 조합 우선
2순위: 동귀어진 (defCP == atkCP) — 상대가 고가치 타겟일 때만
포기: 질 수밖에 없으면 공격 안 함
```

#### 1-3. 공격 순서 최적화 (그리디 매칭)
- 모든 (공격자, 방어자) 조합의 유효 CP 차이를 계산
- **가장 유리한 매칭부터** 공격 실행 (속성 상성 활용 극대화)
- 이미 매칭된 공격자/방어자는 제외하고 다음 매칭 진행

### Phase 2: 소환 전략 개선 (planMainPhase)

**파일**: `BasicAI.swift` — `planMainPhase()`, `performMainPhase()`

#### 2-1. 상대 필드 대응 소환
- 상대 몬스터 속성을 분석하여 **상성 유리한 몬스터 우선 소환**
- 예: 상대 필드에 풍(風) 몬스터 → 화(火) 몬스터 소환 우선도 UP

#### 2-2. 지형 시너지 소환 위치
- 빈 슬롯 중 **같은 속성 지형이 있는 슬롯** 우선 배치
- 속성 지배(3+) 달성 가능하면 해당 속성 몬스터 우선

#### 2-3. 소환 우선도 점수 공식
```
점수 = 기본CP
     + 상성 보너스 (상대 필드에 약점 속성 몬스터 있으면 +500)
     + 지형 매칭 보너스 (같은 속성 지형 슬롯 있으면 +200)
     + 속성 지배 기여 보너스 (지배 달성 임박하면 +300)
```

### Phase 3: 기세 스킬 활용

**파일**: `BasicAI.swift` (새 메서드 추가), `GameViewModel.swift` (AI 턴에서 호출)

#### 3-1. 전투 전 기세 스킬 판단
메인 페이즈 끝에서 사용 여부 판단:

| 스킬 | 비용 | AI 사용 조건 |
|------|------|-------------|
| 투지 | 3 | 핵심 몬스터가 상대보다 약간 약할 때 → +500으로 역전 가능 |
| 전선 돌파 | 6 | 아군 몬스터 3체 이상 |
| 기세 폭발 | 8 | 상대 약한 몬스터(CP ≤ 800) 2체 이상 |

#### 3-2. 기세 보존 판단
- 기세를 소환 비용으로 쓸지 vs 스킬로 쓸지 판단
- 소환이 급하지 않고 기세 ≥ 6이면 스킬용으로 보존

### Phase 4: 드로우 판단 개선

**파일**: `BasicAI.swift` — `chooseDrawCard()`, `evaluateCard()`

#### 4-1. 보드 상태 반영 드로우
```
점수 = 기본 카드 가치
     + 속성 상성 보너스 (상대 필드에 약점 있으면 +300)
     + 패 부족 보충 (몬스터 0장이면 몬스터 +500)
     + 지형 시너지 (현재 지형과 맞으면 +200)
```
